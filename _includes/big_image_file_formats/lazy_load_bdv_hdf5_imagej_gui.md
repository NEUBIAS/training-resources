- Use [ Plugins > Utilities > Monitor Memory... ] to keep an eye on the memory
- Use [ Plugins › BigDataViewer › Open XML/HDF5 ] to open the image
- Use `Shift X, Y, Z` to view orthogonal planes

- Use the arrow keys to zoom
